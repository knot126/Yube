Smash Hit Modding — Custom lights test 2022-08-09 #smashhit

Uploaded 2022-08-09 by Knot126 (@knot126)

Video Information

This is a demo of a new quick test feature for Smash Hit Blender Tools! Versions 2.0.1 and later will now create an HTTP server for you that can be used with a modded version of Smash Hit to allow quick testing of new segments, somewhat like the builtin editor had.

It's still a bit messy to set up perfectly, but it should make editing segments and testing them take a lot less time. Hopefully this will encourage the more impatient creators to make segments and help eliminate the pain of waiting for your APK file to pack.

Right now, only the default exporter settings are working. If you still use stone hack, then it won't work in the way that you might want it to. Templates only work if you have APK Editor Studio open with an apk loaded. Mesh baker settings are always the default.
